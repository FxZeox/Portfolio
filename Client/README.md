# Animated Portfolio

This is a React/Vite based portfolio adhering to specifications provided.

## Setup

```bash
# from workspace root
npm install
npm run dev
```

## Features

- Landing with typing animation
- About, Skills, Projects, Experience, Contact sections
- Dark/light mode toggle
- Animations with Framer Motion, AOS
- Contact form using EmailJS (replace placeholders with your IDs)

## Deployment

Build with `npm run build` and deploy to Vercel, Netlify, GitHub Pages, etc.

## Notes

The project structure follows the recommended folder layout from requirements.
Feel free to add more sections, styles, and replace sample data with real
content. Ensure to configure EmailJS before using the contact form.
